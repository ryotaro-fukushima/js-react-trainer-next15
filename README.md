# JS/React Daily Trainer (Next.js 15 + App Router + TypeScript)

## セットアップ
```bash
npm i
npm run dev
# http://localhost:3000
```

## 型チェックのみ
```bash
npm run typecheck
```

## 収録ページ
- `/es6` ES6+ 構文
- `/functions` 関数・クロージャ
- `/arrays` 配列メソッド
- `/objects` オブジェクト操作
- `/async` 非同期（async/await, Promise.all, try/catch）
- `/events` イベント（preventDefault/stopPropagation, バブリング/キャプチャ）
- `/this` Arrow 関数 vs function の this
- `/types` TypeScript の型テスト（`lib/type-utils.ts` を利用）

## 備考
- 進捗は `localStorage` に保存されます。
- Tailwind を同梱しています（`app/globals.css`）。
